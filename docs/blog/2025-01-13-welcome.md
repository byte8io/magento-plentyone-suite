# Welcome to Mage2Plenty Release Notes

Welcome to the official release notes and changelog for **Mage2Plenty** - your enterprise-grade integration between Magento 2 and PlentyONE.

This blog serves as your central hub for all Mage2Plenty updates, including:

- 🚀 **New Releases** - Major and minor version announcements
- ✨ **New Features** - Exciting enhancements and capabilities
- 🐛 **Bug Fixes** - Important fixes and improvements

## Stay Updated

We maintain a **bi-weekly release cycle** to ensure you always have access to the latest features, bug fixes, and security patches.

### How to Stay Informed

1. **Subscribe to Email Notifications** - Get release updates directly in your inbox at [api.byte8.io/notify/subscribe/mage2plenty](https://api.byte8.io/notify/subscribe/mage2plenty)
2. **Watch on GitHub** - Follow releases on our [GitHub repository](https://github.com/byte8/mage2plenty)
3. **Check Regularly** - Visit this page for the latest updates
4. **Review Changelogs** - Read the detailed [Changelog](/docs/changelog) for complete version history

## Current Version

The latest stable version is **1.15.x** with full support for:
- Magento 2.4.4 - 2.4.8+
- PHP 8.1, 8.2, 8.3, 8.4
- Multi-Source Inventory (MSI)
- Message Queue Processing
